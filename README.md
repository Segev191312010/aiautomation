# AIAutomation

## Overview
This repository is intended for automation projects using AI.

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/Segev191312010/aiautomation.git
   ```
2. Navigate to the project directory:
   ```bash
   cd aiautomation
   ```
3. Install required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the automation scripts as needed.
